# Product Overview

## AI-Enhanced Dungeon Master Game

This project recreates the classic 1987 Dungeon Master experience with modern AI enhancements. The game combines traditional grid-based dungeon crawling with intelligent AI systems that learn player behavior and provide adaptive assistance.

### Core Features

- **Classic Dungeon Crawler**: Grid-based movement, real-time combat, retro pixel art aesthetics
- **AI Mentor System**: Learns player behavior patterns and provides contextual tactical guidance
- **Visual Adaptation Engine**: Adjusts UI complexity and styling based on player skill level
- **Combat Analysis System**: Provides detailed post-combat analysis with efficiency scoring (0-100%)
- **Adaptive Learning**: AI systems persist learning data across gaming sessions

### Target Experience

The game maintains authentic retro gameplay while adding modern AI features that enhance learning and strategy development. Players experience classic dungeon crawling with intelligent assistance that adapts to their playstyle without disrupting the core game mechanics.

### Key Differentiators

- AI systems that learn and adapt to individual player behavior
- Real-time thought bubble UI showing AI decision-making process
- Performance-based visual adaptation that grows with player skill
- Comprehensive combat replay system with optimal strategy analysis
